customShapes Classes
--------------------
Create your own shape or chords
The index of each list corresponds to each other, meaning the elements at the same index in each list represent the attributes of the same note, position, or scale degree.

shapeName1: {
    note: [list of notes],
    position: [list of positions],
    scaleDegree: [list of scale degrees]
},
shapeName2: {
    note: [list of notes],
    position: [list of positions],
    scaleDegree: [list of scale degrees]
},
